Environmental Health Map Design

An Experience in Sketching, User Testing, and Divergent Iteration

Learn More

The Problem

Existing environmental health maps are slow, difficult to use, and lack sufficient data. The goal of this project is to develop a design for an environmental health map of the U.S. that anyone can use to see environmental factors in their community.

Constraints and the Task

This project was highly constrained by time. My work has been developing sketches for the site and preparing them for user testing. These tests will inform future design of the product, establishing which design styles to create hi-fi prototypes of and conduct usability tests on.

Design 1

Design 1 has the map take up the entire screen, with boxes on the side containing the checkboxes for different layers. Testing on this design will demonstrate user interest in a map which covers the entire browser window.

Design 2

Design 2 has the map only be a small square window next to background information. This tests how much users favor background information and clear options but a smaller map window.

Design 3 (1/2)

Design 3 has the map take up the entire screen but the options are hidden under a dropdown menu. This allows users to see the map unobscured on the entire screen, but it may be unintuitive to select filters.

User testing

I intentionally used very rough sketches to conduct user testing to encourage more honest feedback. Users will be asked to explain completing a different task for each interface, and their actions and speed will be recorded, though speed will only be measured across attempt order (first attempts with other first attempts, ect.). After the task, users will be asked the following questions:

  • What did you think of the interface?

  • What did you think of the layout of the content?

  • What was the best thing about using this site?

  • What was the worst thing about using this site?

  • How would you improve the interface?

  • How would you rate using this interface out of 10?

Future Work

The results of the early user testing will inform future design of this project. I plan to build hi-fidelity prototypes of the most successful designs and conduct a series of usability tests to converge on a final design. The complete process will entail a few more stages of prototyping and testing.
